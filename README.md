# IMC Calculator 🧮

BMI calculator based on the Mexican official health standard.

## Features
- Asks if the user knows what BMI is and explains it if needed
- Calculates BMI from weight (kg) and height (m)
- Classifies the result: underweight, normal, overweight or obese
- Loop to perform multiple calculations

## How to run
python imc_calculator.py

## Built with
- Python 3
- While loops, conditionals, type conversion and arithmetic operations

## Note
This is an educational project. BMI is just one indicator. Always consult a healthcare professional for accurate health assessments.
